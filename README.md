## How Start

1. Create App

```bash
    npm npx create-next-app@latest app_name
```

2. install shadcn

```bash
    npx shadcn@latest init
```

3. install shadcn

```bash
    npx shadcn@latest add radio-group label checkbox button
```

4. install Prisma

```bash
    npm i --dev prisma @prisma/client
```

5. Init Prisma

```bash
    npx prisma init
```

6. migrate Prisma

```bash
    npx prisma migrate dev
```

7. generate Prisma

```bash
    npx prisma generate dev
```

8. generate Prisma

```bash
    npx prisma studio
```
