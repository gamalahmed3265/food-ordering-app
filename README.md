## How Start

1. Create App

```bash
    npm npx create-next-app@latest app_name
```

2. install shadcn

```bash
    npx shadcn@latest init
```

3. install shadcn

```bash
    npx shadcn@latest add radio-group label checkbox button
```

4. install Prisma

```bash
    npm i --dev prisma @prisma/client
```

5. Init Prisma

```bash
    npx prisma init
```

6. generate Prisma

```bash
    npx prisma generate dev
```

7. migrate Prisma

```bash
    npx prisma migrate dev
```

8. generate Prisma

```bash
    npx prisma studio
```

9. negotiator

```bash
    npx i negotiator
    npx i @types/negotiator
```

10. install bcrypt

```bash
    npx i bcrypt
    npx i @types/bcrypt
```

11. install zod

```bash
    npx prisma zod
```

12. install zod

```bash
    npx shadcn@latest add toast
```
